Mac - 終端機常用快速鍵

2020-03-25 1 min read # Mac
  1. Control + A 移動到開頭
  2. Control + E 移動到結尾
  3. Control + P 在終端機可叫出上一個執行的指令
  4. Control + N 在終端機可叫出下一個執行的指令
  5. Control + B 往前移動一個字元
  6. Control + F 往後移動一個字元
  7. Control + W 刪除一個單字
  8. Control + U 刪除整行
  9. Control + K 刪除游標後至行尾的所有文字
