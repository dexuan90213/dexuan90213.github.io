flex

主軸：main-axis
次軸：cross-axis

父層設定 flex

  • 當父層設定成 display: flex 後，此層就會變成 flex-item 物件

  • flex-item：具有彈性的空間

  • 子層預設會是橫向排列

  • 資料預設排列，由上至下 由左至右

justify-content（對齊與分佈）

  1. flex-start 對齊資料流主軸起點
  2. flex-end 對齊資料流主軸終點
  3. center 對齊資料流主軸中間
  4. space-around 將剩餘空間分配至每一個物件的左右兩側
  5. space-between 將剩餘空間分配至物件跟物件之間
  6. space-evenly 將剩餘空間分配至每一個物件左右，任兩個物件之間距離以及物件到邊緣的距離相等

flex-wrap（換列）

  1. nowrap 不換列（預設值）
  2. wrap 換列
  3. wrap-reverse 換行反向 (換行的方向 也就是次軸的方向)

flex-direction（主軸方向）

  1. row 資料流由左至右（預設值）
  2. row-reverse 資料流由右至左
  3. column 直向 資料流由上而下
  4. column-reverse 直向 資料流由下而上

小提示：flex-direction flex-wrap 可以簡寫成 flex-flow: row wrap;

align-items

控制彈性列裡面的項目要對齊該列的起點、終點、中間

在設定 flex 後，且有設定 wrap 則會自動分割成多個列，如果是直像的話，則會有多個欄（具有彈性的列）

  1. stretch (預設): 填滿整個容器 (min-width/max-width 不會被蓋掉)
  2. flex-start 對齊容器內次軸的起點
  3. flex-end 對齊容器內次軸的終點
  4. center 對齊次軸的中間
  5. baseline 容器內元素之間互相對齊基線

align-self

單獨控制彈性列裡面的物件的對齊
屬性同 align-items

align-content

當有換列的情狀時，且次軸有剩餘的空間，設定彈性列（欄）的對齊與分配方式，不適用於只有單行的 flex-item。

  1. flex-start 容器對齊次軸的起點
  2. flex-end 容器對齊次軸的終點
  3. center 容器對齊次軸的中間
  4. space-between 將剩餘空間分配至容器與容器之間
  5. space-around 將剩餘空間分配至容器的兩旁
  6. space-evenly 將剩餘空間分配至容器之間，但容器與容器以及容器至邊緣的距離相等
  7. stretch（預設值）拉伸容器佔滿剩餘的空間

子層設定

flex-grow

延展，將剩餘空間分配幾份給設定的物件

flex-shrink

收縮，將超過父層的空間，除以要收縮幾份後，將設定的物件扣除要收縮的份數

flex-basis

設定物件為主軸的長度，橫向時為寬，直向時為高度
此設定會蓋掉 width（橫向） 或 height（直向）

上述可簡寫成：

flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

The second and third parameters (flex-shrink and flex-basis) are optional. Default is 0 1 auto.
