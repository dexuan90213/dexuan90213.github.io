建立第一個 Rails 應用程式

2020-03-25 1 min read # Rails # Ruby # Ruby on Rails

建立 Rails 專案

$ rails new hello_rails

將使用目前所安裝最新（目前所安裝的是 6.0.1）的 Rails 產生一個名為 hello_rails 的專案

進入專案目錄

$ cd hello_rails

啟動 rails server

$ rails server

或簡寫成

$ rails s

查看首頁

瀏覽器打開 http://localhost:3000
看到歡呼畫面表示成功執行應用程式

產生特定版本的 Rails 版本的專案

$ rails _5.2.3_ new project

javascript 資料夾不是只能放 js 檔案，可放前端的檔案 webpacker
adapter: 轉接器，對轉接頭撈資料，轉接頭幫忙跟資料庫溝通撈資料
資料庫分成三個，因不同環境 有不同的資料，test 不應該影響到 production 資料庫

bundler -> Gemfile 套件打包器

bundle install -> 產生 Gemfile.lock

Gemfile

~> 安裝 約在 6.0.1 的版本 6.0.2 6.0.3 不會安裝 6.1.0
