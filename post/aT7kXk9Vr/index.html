Mac - 加速 Time Machine 備份速度

2020-03-25 1 min read # Mac # macOS # Time Machine # 時光機

macOS 的 Time Machine 有限流，避免備份時影響到正在執行的工作。

解除方式也很簡單，確定程式都關掉、Time Machine 也沒有正在備份中，打開終端機程式，輸入下面指令

sudo sysctl debug.lowpri_throttle_enabled=0

完成後再次啟動 Time Machine 備份任務，這時會發現備份速度變快很多，等待備份完畢後，重新開機或是執行下面指令，即可恢復至原先有限流的狀態：

sudo sysctl debug.lowpri_throttle_enabled=1
