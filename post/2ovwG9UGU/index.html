安裝 Ruby 及 Ruby on Rails

2020-03-25 4 min read # Rails # Ruby # Ruby on Rails

安裝 Ruby

使用 rvm 套件管理器安裝 Ruby

  1. 不需要 root 權限
  2. 可切換不同版本的 Ruby
  • 查詢是否內建 Ruby
$ which ruby
# > /usr/bin/ruby # 系統內建的 Ruby
  • 安裝 rvm 官網
$ rvm list # 查看 rvm 已經安裝的 Ruby 版本
$ rvm list known # 列出目前可以安裝的 Ruby 版本有哪些
$ rvm get master # 更新 list known 列表
$ rvm -v # 顯示 rvm 版本
  • 使用 rvm 安裝 Ruby
$ rvm install 2.6.5 # 安裝 Ruby 2.6.5 版本
  • 切換 Ruby 版本
$ rvm use 2.6.3 # 使用 2.6.3 版本
  • 設定某個版本為預設值
    下次開啟終端機後，執行 Ruby 就是所指定的版本號了
$ rvm user 2.6.3 --default
  • 移除特定的版本
$ rvm uninstall 2.6.3

安裝 Ruby on Rails

$ gem install rails

會到 Rubygems.org 安裝最新版本的 Rails

安裝特定版本的 Rails

$ gem install rails -v 5.2.3

參數 -v 指定要安裝的版本

安裝嚐鮮版本的 Rails

$ gem install rails --pre

檢查 Rails 目前版本

$ rails -v 
# Rails 6.0.1

在 Rails 5.2.3 專案底下執行 rails -v

$ rails -v 
# Rails 5.2.3

rails -v 會去上層找 有沒有 bin/rails 資料夾或檔案，當找到專案的 bin/rails，則執行該專案所使用的 rails 版本 ，如果都沒有，則執行目前所安裝最新版本的 Rails

Rails 6 要多安裝 yarn

  • Rails 6 預設啟用 webpacker 作為 JavaScript 文件管理方案
  • 在建立專案前先確認有無安裝 yarn
  • 使用 $ yarn -v 如果沒有顯示版本號，則要先安裝

各個平台的安裝方式參考官網
