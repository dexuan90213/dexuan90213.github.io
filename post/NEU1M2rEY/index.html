類別 與 實體

建立 class

class Cat
  def eat(food)
    puts "#{food} is good"
  end
end

kitty = Cat.new
kitty.eat('banana')

類別命名時 必須是常數

繼承

把共同特徵放在同一個 class 裡面

class Animal
  def walk(animal)
    puts "#{animal} walk."
  end

  def eat(animal)
    puts "#{animal} eat."
  end
end

class Dog < Animal
end

class Cat < Animal
end

dog1 = Dog.new
cat1 = Cat.new

dog1.walk "123"
cat1.walk "123123"

dog1.eat "123"
cat1.eat "123123"

初始化 initialize

class Cat
  def initialize(name)
    puts "hi #{name}"
  end
end

cch = Cat.new('cch') # new 後面的引數是 initialize 的參數

沒有寫 initialize

當 Cat 沒有寫繼承其他，會自動繼承自 Object，Object 的 initialize 沒有參數，所以會出現參數數量錯誤

class Object
  def initialize
  end
end

class Cat < Object
end

cch = Cat.new('cch', 32)

實體變數

class Cat
  def initialize(name, age)
    @name = name
    @age = age
  end

  def greeting
    puts "I am #{@name}, age: #{@age}"
  end
end

kitty = Cat.new('kitty', 32)

kitty.greeting

new 跟 initialize

new 後面的引數，在 new 後，會帶入 initialize 的參數

實體方法 = 作用在實體上的方法
類別方法 = 做用在類別上的方法，方法名稱前面加 self

class Cat
  def self.all
    puts "全部的貓兒"
  end
end
